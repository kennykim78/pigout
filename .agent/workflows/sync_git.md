---
description: 피그아웃 프로젝트 진입 시 Git 최신 상태 확인 및 동기화
---

이 워크플로우는 프로젝트에 처음 진입하거나 세션을 시작할 때 실행되어 최신 코드를 유지하도록 돕습니다.

1. 원격 저장소의 최신 변경 사항을 가져옵니다.
   // turbo
   run_command("git fetch")

2. 로컬 브랜치와 원격 브랜치의 상태를 비교합니다.
   // turbo
   run_command("git status -uno")

3. 만약 로컬이 뒤처져 있다면, 현재 작업 중인 변경 사항을 보관(stash)하거나 커밋한 후 업데이트를 진행합니다.

- 변경 사항이 없다면 바로 `git pull`을 수행합니다.
- 변경 사항이 있다면 사용자에게 확인 후 진행합니다.
